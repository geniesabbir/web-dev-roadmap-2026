# Days 6-7: Job Search Strategy & Salary Negotiation

## Introduction

The job search is a numbers game with a strategy component. Success requires a systematic approach to finding opportunities, networking effectively, and ultimately negotiating the best possible offer. These two days cover everything from sourcing job leads to closing a deal that reflects your true value.

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    Job Search Funnel                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   Applications            100                                           │
│        │                                                                │
│        ▼                                                                │
│   Recruiter Screens       20  (20% conversion)                         │
│        │                                                                │
│        ▼                                                                │
│   Technical Screens       10  (50% conversion)                         │
│        │                                                                │
│        ▼                                                                │
│   On-sites                 5  (50% conversion)                         │
│        │                                                                │
│        ▼                                                                │
│   Offers                   2  (40% conversion)                         │
│        │                                                                │
│        ▼                                                                │
│   Accepted Offer           1                                           │
│                                                                         │
│   Key insight: Expect ~50-100 applications for 1-2 offers              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## Learning Objectives

By the end of these two days, you will:
- Build a systematic job search strategy
- Leverage networking to get referrals
- Navigate the interview process effectively
- Negotiate salary and benefits confidently
- Evaluate and compare offers

## Day 7: Negotiation & Closing

### 1. Understanding Compensation

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    Total Compensation Components                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  BASE SALARY                                                            │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  • Guaranteed cash, paid regularly                               │   │
│  │  • Typically reviewed annually                                   │   │
│  │  • Most negotiable for most companies                            │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  ANNUAL BONUS                                                           │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  • Target % of base (e.g., 10-20%)                               │   │
│  │  • Usually tied to performance + company performance             │   │
│  │  • Not guaranteed - may be 0% to 150% of target                  │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  EQUITY / STOCK                                                         │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  RSUs (Restricted Stock Units):                                  │   │
│  │  • Actual shares, vest over time (typically 4 years)             │   │
│  │  • Value = current stock price × number of shares                │   │
│  │                                                                  │   │
│  │  Stock Options (more common at startups):                        │   │
│  │  • Right to buy at "strike price"                                │   │
│  │  • Value = (current price - strike price) × shares               │   │
│  │  • Worth nothing if company value doesn't increase               │   │
│  │                                                                  │   │
│  │  Startup Equity:                                                 │   │
│  │  • High risk, potentially high reward                            │   │
│  │  • Discount value significantly (50-90%)                         │   │
│  │  • Ask: Fully diluted ownership %, last 409A valuation           │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  SIGNING BONUS                                                          │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  • One-time payment when you join                                │   │
│  │  • Often used to offset unvested equity you're leaving           │   │
│  │  • May have clawback if you leave within 1-2 years               │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  BENEFITS                                                               │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  • Health insurance (value: $10-20K/year for family)             │   │
│  │  • 401k match (value: 3-6% of salary)                            │   │
│  │  • PTO (value varies)                                            │   │
│  │  • Remote work flexibility                                       │   │
│  │  • Learning budget, conference budget                            │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  EXAMPLE TOTAL COMP CALCULATION:                                        │
│  Base: $150,000                                                         │
│  Bonus: $22,500 (15% target)                                            │
│  RSUs: $50,000/year (vesting)                                           │
│  ─────────────────────                                                  │
│  Total Comp: $222,500/year                                              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3. Negotiation Strategy

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    Negotiation Framework                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  RULE #1: ALWAYS NEGOTIATE                                              │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  • 70% of employers expect negotiation                           │   │
│  │  • Not negotiating leaves money on the table                     │   │
│  │  • Sets your baseline for future raises                          │   │
│  │  • Won't lose an offer by negotiating professionally             │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  TIMING:                                                                │
│                                                                         │
│  When asked about salary expectations (early in process):               │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  "I'm focused on finding the right fit and am flexible on        │   │
│  │  compensation. I'd love to learn more about the role first.      │   │
│  │  What's the range budgeted for this position?"                   │   │
│  │                                                                  │   │
│  │  If pressed:                                                     │   │
│  │  "Based on my research, similar roles at comparable companies    │   │
│  │  pay in the range of $X-$Y. Is that in line with your budget?"   │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  When you receive an offer:                                             │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  1. Express enthusiasm: "Thank you! I'm excited about this       │   │
│  │     opportunity."                                                │   │
│  │  2. Ask for details in writing                                   │   │
│  │  3. Request time: "Can I have a few days to review the full      │   │
│  │     package?"                                                    │   │
│  │  4. NEVER accept on the spot                                     │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  NEGOTIATION SCRIPT:                                                    │
│  ─────────────────────────────────────────────────────────────────────  │
│  "Thank you for the offer. I'm very excited about joining [Company]    │
│  and working on [specific project/team].                                │
│                                                                         │
│  After reviewing the offer and researching market rates, I was          │
│  hoping we could discuss the base salary. Based on my experience        │
│  with [relevant skill/achievement] and the going rate for similar       │
│  roles, I was expecting something closer to $X.                         │
│                                                                         │
│  Is there flexibility to move closer to that number?"                   │
│  ─────────────────────────────────────────────────────────────────────  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 4. What to Negotiate

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    Negotiation Levers                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  IN ORDER OF TYPICAL FLEXIBILITY:                                       │
│                                                                         │
│  1. SIGNING BONUS (Most Flexible)                                       │
│     • One-time cost, easier for companies to approve                    │
│     • Good for offsetting equity you're leaving                         │
│     • Ask: "Is there room for a signing bonus?"                         │
│                                                                         │
│  2. BASE SALARY                                                         │
│     • Aim for 10-20% above initial offer                                │
│     • Has compound effect on future raises                              │
│     • Ask: "Is there flexibility on the base?"                          │
│                                                                         │
│  3. EQUITY / RSUs                                                       │
│     • May have more room than salary at tech companies                  │
│     • Ask for more shares or accelerated vesting                        │
│     • Ask: "Can we increase the equity component?"                      │
│                                                                         │
│  4. START DATE                                                          │
│     • Get time off before starting                                      │
│     • Negotiate later start to overlap with other offers                │
│                                                                         │
│  5. TITLE / LEVEL                                                       │
│     • Affects future opportunities and comp                             │
│     • Ask: "Based on my experience, would Senior level be appropriate?" │
│                                                                         │
│  6. REMOTE / HYBRID FLEXIBILITY                                         │
│     • Increasingly negotiable post-pandemic                             │
│     • High value, low cost to employer                                  │
│                                                                         │
│  7. PTO                                                                 │
│     • Some companies will add extra days                                │
│     • Especially if matching previous employer                          │
│                                                                         │
│  8. RELOCATION                                                          │
│     • If applicable, can be significant ($10-50K)                       │
│                                                                         │
│  9. LEARNING BUDGET                                                     │
│     • Conference attendance, courses                                    │
│     • Easy win, small cost to company                                   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

```
│  ┌─────────────────────────────────────────────────────────────────┐   │

│  │  • Mention you have competing offers                             │   │

│  │  • Express genuine interest in the role                          │   │
│  │                                                                  │   │
│  │  DON'T:                                                          │   │
```

## Handling Rejections

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    Dealing with Rejection                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  MINDSET:                                                               │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  • Rejection is normal - even great candidates get rejected      │   │
│  │  • It's often about fit, timing, or luck - not your abilities    │   │
│  │  • Each interview is practice for the next one                   │   │
│  │  • You only need ONE yes                                         │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  ASK FOR FEEDBACK:                                                      │
│  ─────────────────────────────────────────────────────────────────────  │
│  "Thank you for letting me know. I really enjoyed learning about       │
│  [Company]. If possible, I'd appreciate any feedback on my             │
│  interviews that might help me improve. I understand if you're         │
│  not able to share specifics."                                         │
│  ─────────────────────────────────────────────────────────────────────  │
│                                                                         │
│  LEARN AND ADJUST:                                                      │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  • Note which stage you were rejected                            │   │
│  │  • If coding: More practice needed                               │   │
│  │  • If system design: Study specific areas                        │   │
│  │  • If behavioral: Refine your stories                            │   │
│  │  • If resume screen: Improve resume/target better                │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  KEEP THE DOOR OPEN:                                                    │
│  ─────────────────────────────────────────────────────────────────────  │
│  "I'd love to stay in touch and be considered for future               │
│  opportunities. Would it be okay if I reached out in 6-12 months?"     │
│  ─────────────────────────────────────────────────────────────────────  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    Job Search Success Checklist                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  FINDING OPPORTUNITIES:                                                 │

│  ✓ Optimize LinkedIn for searchability                                 │
│  ✓ Track all applications systematically                               │
│                                                                         │
│  NETWORKING:                                                            │
│  ✓ Reach out to 5-10 people per week                                   │
│  ✓ Make it easy for people to refer you                                │
│  ✓ Follow up (once) if no response                                     │
│                                                                         │
│  NEGOTIATION:                                                           │
│  ✓ Always negotiate (professionally)                                   │
│  ✓ Research market rates thoroughly                                    │
│  ✓ Never accept on the spot                                            │
│  ✓ Consider total compensation, not just base                          │
│  ✓ Get everything in writing                                           │
│                                                                         │
│  MINDSET:                                                               │
│  ✓ It's a numbers game - expect ~50 applications per offer             │
│  ✓ Rejection is normal and doesn't reflect your worth                  │
│  ✓ You only need one yes                                               │
│                                                                         │
```
