# Phase 5: Advanced Topics & Job Prep

**Duration:** 8-10 weeks
**Goal:** Master advanced concepts and prepare for professional work

## Overview

This final phase covers advanced topics that separate good developers from great ones, plus everything you need to land your first (or next) developer role.

## Sections

### 1. [System Design](./01-system-design/README.md) (2-3 weeks)
- Scalability fundamentals
- Database scaling
- Caching strategies
- Load balancing
- Microservices vs Monolith
- Message queues
- Design patterns

### 2. [Performance Optimization](./02-performance/README.md) (2 weeks)
- Frontend performance (Core Web Vitals)
- Backend performance
- Database optimization
- Caching strategies
- Profiling and debugging

### 3. [AI Integration](./03-ai-integration/README.md) (2 weeks)
- LLM APIs (OpenAI, Anthropic)
- Building AI features
- Prompt engineering
- RAG applications
- AI-powered search

### 4. [Job Preparation](./04-job-prep/README.md) (2-3 weeks)
- Portfolio building
- Resume writing
- Interview preparation
- System design interviews
- Behavioral interviews
- Salary negotiation
