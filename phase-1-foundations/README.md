# Phase 1: Foundations

**Duration:** 8-10 weeks
**Goal:** Master the core building blocks of web development

## Overview

This phase covers the essential skills every web developer needs. Don't rush through this - a strong foundation makes everything else easier.

## Sections

### 1. [HTML & CSS](./01-html-css/README.md) (2-3 weeks)
- Semantic HTML5
- CSS Fundamentals
- Flexbox & Grid
- Responsive Design
- CSS Architecture (BEM)
- Accessibility basics

### 2. [JavaScript](./02-javascript/README.md) (3-4 weeks)
- Core syntax and data types
- Functions and scope
- DOM manipulation
- Events and event handling
- Async JavaScript (Promises, async/await)
- ES6+ features
- Error handling

### 3. [TypeScript](./03-typescript/README.md) (2 weeks)
- Types and interfaces
- Generics
- Type narrowing
- Utility types
- TypeScript with DOM
- Configuration

### 4. [Git & GitHub](./04-git/README.md) (1 week)
- Git fundamentals
- Branching strategies
- Pull requests
- Collaboration workflows
- Commit conventions

## Phase 1 Project

**Build:** A responsive personal portfolio website
- Mobile-first design
- Semantic HTML
- Modern CSS (Grid/Flexbox)
- JavaScript interactions
- TypeScript refactor
- Hosted on GitHub Pages

## Tips for Success

1. **Type everything** - Don't copy-paste code
2. **Break things** - Experiment and see what happens
3. **Use DevTools** - Inspect everything
4. **Read MDN** - The best documentation
5. **Build small projects** - Apply concepts immediately
